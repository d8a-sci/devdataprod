A fuel economy surveyor

E.N.
September 2015

Idea

Create a basic Shiny app on the ggplot2 mpg data set where the user can filter on:

  1. Transmission type (select at least 1)
    • Manual
    • Automatic
  2. Traffic type (select only 1)
    • City traffic
    • Highway travel

The total average fuel economy will be calculated.

Example

# library calls & preprocessing
# omitted for brevity
filter(dt, trans=="auto") %>% 
        group_by(year)    %>%
        summarise(mean(hwy))

Source: local data table [2 x 2]

   year mean(hwy)
  (int)     (dbl)
1  1999  22.13514
